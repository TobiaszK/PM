BookCorp

This is application which enables to organize:
  • work
  • flow of information
  • plans
  • files among other participants via internet
Check which functionalities will be in use:
Overview This is a wall where everything appears if it was added. The visibility for particular users is the same as decided during creating the item by creator. Each category has its color and each "Activity" has its own sign. Admins can also change the project-status there.
User Admin- can add/remove users and whole content
Programmer- can add new content
Guest- if accepted by user, can watch what is happening in the project
Message Messages can be created with chosen recivers. They can be commented and some basic categories can be added(Each category has its own color). The time of sending the messagee will be also visible. Some files can be also uploaded with message
To-dos Topics with title and description where new items with check-boxes are added. The creator can decide who can see and edit topics or/and items created. Topics, items, descriptions can be deleted and comments can be added.
Files Files can be upploaded with some comments, asigned categories and users' visibility.
Search Enables search for of projects by name and status as well as messages by title and category.
Projects Shows all projects in a list and there is an ability to search projects by name
Writeboards This is a concept of a white board where everyone can add and remove something previously written. The visibility also can be managed as well as comments. New page has title and the body.
