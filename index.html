Overview

This is a wall where everything appears if it was added. The visibility for particular users is the same as decided during creating the item by creator. Each category has its color and each "Activity" has its own sign. Admins can also change the project-status there. Such 'general' look helps to see quickly what was done recently

Messages

Messages can be created with chosen recivers. They can be commented and some basic categories can be added(Each category has its own color). The time of sending the messagee will be also visible. Some files can be also uploaded with message. Be in contact with associates.

To-dos

Topics with title and description, where new items with check-boxes are added. The creator can decide who can see and edit topics or/and items created. Topics, items, descriptions can be deleted and comments can be added. Quick look and you know, what has to be done soon and what has left

Writeboards

This is a concept of a white board where everyone can add and remove something previously written. The visibility also can be managed as well as comments. New page has title and the body. Useful during brainstorms via internet, everyone can participate.

Projects

Shows all projects in a list. There is an ability to search for projects by name if you need something, which was done in the past

Users

Admin
can add/remove users and whole content
Programmer
can add new content
Guest
if accepted by user, can watch what is happening in the project

Files

Files can be upploaded with some comments, asigned categories and users' visibility.

Search

Enables search for of projects by name and status as well as messages by title and category.
